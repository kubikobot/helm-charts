# KubikoBot Helm Charts for Kubernetes

This repository holds the index file for [the Kubikobot Helm Repository](https://kubikobot.github.io), served using
[GitHub pages](https://pages.github.com/). The index file serves to hold the list of available helm charts distributed or configured
by Kubikobot, including the associated metadata that describes the charts. You can read more about the repository index
file from the [official documentation](https://helm.sh/docs/developing_charts/#the-index-file).

## How do you install the charts in this Repo?

You can access these charts by adding the Kubikobot Helm Repo to your client:

```bash
$ helm repo add kubikobot https://kubikobot.github.io
```

Then, you can access any of the charts under the `kubikobot` name.
