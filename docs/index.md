# Welcome

Welcome to the documentation for the [Kubiko Helm charts](https://github.com/kubikobot/helm-charts) project.

## Getting started

[Helm](https://helm.sh) must be installed to use the charts in this repository.

Refer to Helm's [documentation](https://helm.sh/docs/) to get started.

## Installation

```sh
helm repo add kubikobot http://kubikobot.github.io/helm-charts/
```

You can then run `helm search repo kubikobot` to search the charts.
